Primality Testing

Posted on 2018-04-27 | In Number-Theoretic Algorithms

Description

A prime number is a natural number greater than 1 that cannot be formed by multiplying two smaller natural numbers. A primality test is an very useful algorithm, especially in cryptography. It tests whether the input integer is prime or not. In order to test a very huge integer within reasonable time, a high-performance primality testing algorithm is worth to be studied. For simplicity, we will use $n$ to represent the number needs to be tested. $\left( {n > 1} \right)$

Trial Division Method (Native)

A simple method is to check whether any integer $d$ that $2 \le d \le n - 1$ divides $n$ when $n$ larger than $2$. If n is $2$, then we know $2$ is a prime.

Trial Division Method (Root)

Due to the fact that if $d$ is a divisor of $n$ bigger than or equal to $\sqrt n $, then there must be a corresponding divisor $\frac{n}{d}$ of $n$ smaller than or equal to $\sqrt n $. We can easily prove that:

$$d \ge \sqrt n $$

$$\frac{n}{d} \le \frac{n}{ {\sqrt n } } = \sqrt n $$

So we needless to check all the integers $d$ from $2$ to $n-1$. We only test all the integer $d$ that $2 \le d \le \sqrt n $.

Trial Division Method (Elementary Sieve)

Normally we need to test whether $n​$ is divdable by any integer $d​$ that $2 \le d \le \sqrt n ​$. However, we noticed that $d​$ can be written as $6k - 1​$, $6k​$, $6k+1​$, $6k+2​$, $6k+3​$ or $6k+4​$ ($k​$ is an integer). Since once $n​$ can be divided by $6k​$, $6k+2​$, $6k+3​$ or $6k+4​$, than $n​$ must be divided by $2​$ or $3​$. So first we check if $n​$ ($n \ne 2​$ or $3​$) can be divided by $2​$ or $3​$. If it does, then $n​$ is a composite number. Otherwise we need to check whether $n​$ is divdable by any integer $d​$ in the remained form of $6k \pm 1​$ $(2 \le d \le \sqrt n )​$.
