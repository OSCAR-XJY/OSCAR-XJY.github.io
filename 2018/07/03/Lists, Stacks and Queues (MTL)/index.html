Lists, Stacks and Queues (MTL)

Posted on 2018-07-03 | In My Template Library

Description

Lists, Stacks and Queues are collections of data whose orders are not linearly placed in the physical memory, but can be manipulated by utilizing pointers. Each of these three data structures has its especial superiority which may greatly improve the speed of our programs. By building our own versions of these three data structures, we can practice coding and may be able to explore the ingenious designs of STL.
